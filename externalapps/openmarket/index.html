🚀 How to Publish Your App

  • Have a valid Discord and GitHub account
  • Your app must be hosted on a public GitHub repository
  • Join the official OpenMarket Discord server
  • Submit your repo link in the #app-suggestions channel
  • Wait for a DM from the moderation team

❌ Rejection Reasons

  • NSFW content
  • Security risks or suspicious behavior
  • High dependency requirements
  • Constant microtransaction prompts
  • Hacking, DDoS, or malicious tools
